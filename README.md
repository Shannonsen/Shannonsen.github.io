# Shannon Sen Perdomo - Portfolio Personal

Portfolio personal construido con React, TypeScript y Hopper Design System.

🌐 **Sitio en vivo**: [https://shannonsen.github.io](https://shannonsen.github.io)

## 🚀 Tecnologías Utilizadas

- **React 19** - Librería de UI
- **TypeScript 5.9** - Tipado estático
- **Vite** - Build tool y dev server
- **Hopper Design System** - Sistema de diseño de Workleap
- **Embla Carousel** - Carousel moderno y accesible
- **CSS3** - Estilos personalizados

## 📦 Instalación

```bash
# Instalar dependencias
npm install
```

## 🛠️ Desarrollo

```bash
# Iniciar servidor de desarrollo
npm run dev
```

El proyecto estará disponible en `http://localhost:5173`

## 🏗️ Build

```bash
# Crear build de producción
npm run build

# Preview del build
npm run preview
```

## 🚀 Deploy a GitHub Pages

Para desplegar a GitHub Pages:

```bash
# 1. Crear el build
npm run build

# 2. Agregar cambios a git
git add .
git commit -m "Deploy portfolio"

# 3. Push a GitHub
git push origin main
```

Luego, en la configuración del repositorio en GitHub:
1. Ve a Settings > Pages
2. En "Source", selecciona "Deploy from a branch"
3. Selecciona la rama `main` y la carpeta `/dist`
4. Guarda los cambios

O puedes usar GitHub Actions para deployment automático.
